## 📁 ファイル一覧

### 要件定義.md

システムの詳細な要件定義です。最新の仕様が記載されています。

### 画像取得システム.md
**現行仕様（As-Is）**

現在のシステムの仕様を記載したドキュメントです。

### 開発ナレッジ.md
**開発・検証過程の知見**

開発・検証過程で得られた知見をまとめたドキュメントです。次回以降の開発・修正時に参考にしてください。

**内容**:
- 問題発見と原因分析
- アプローチ比較と意思決定
- 実装過程と課題
- 最終的な解決策
- システム設計の決定
- 今後の開発・修正時の注意点

### フォルダ構造説明.md
**フォルダ構造の説明**

新しいフォルダ構造の説明と、旧システムとの違いを記載したドキュメントです。

## 💡 使用方法

### 開発・修正時

1. **`開発ナレッジ.md`** を参照
   - 過去の問題と解決策を確認
   - 新しいパターンの追加方法を確認
   - デバッグ時のポイントを確認

2. **`要件定義.md`** を参照
   - 最新の仕様を確認
   - 実装すべき機能を確認

3. **`画像取得システム.md`** を参照
   - 現行の実装状況を確認

### 使用方法の確認

- **`../README.md`** を参照（ルートのREADME.md）
